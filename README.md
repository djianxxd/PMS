# 🏃‍♂️ 自律人生 - 个人生活管理系统

一个功能完整的个人生活管理Web应用，帮助您记录、管理和改善日常生活习惯。

## ✨ 功能特性

### 🏠 仪表盘
- **数据概览** - 直观展示生活各项统计数据
- **连续打卡** - 显示当前连续打卡天数
- **完成率统计** - 待办事项完成情况
- **快速操作** - 便捷访问各功能模块
- **动态统计** - 实时更新的收入支出统计
- **习惯进度** - 可视化习惯完成情况

### 💰 收支管理
- **交易记录** - 详细记录每笔收入和支出
- **分类管理** - 预设丰富的收支分类（餐饮、交通、娱乐等）
- **自定义分类** - 支持用户自定义添加个人分类
- **月度统计** - 按月查看收支汇总
- **预算目标** - 设置和跟踪预算目标
- **数据分析** - 收支趋势和消费分析

### ✅ 习惯打卡
- **习惯创建** - 建立个人习惯目标
- **每日打卡** - 简单快捷的打卡记录
- **连续天数** - 自动计算习惯连续坚持天数
- **完成统计** - 总打卡天数和今日状态
- **习惯日历** - 日历视图展示打卡历史
- **进度可视化** - 直观的进度条显示

### 📋 日程待办
- **任务管理** - 创建、编辑、删除待办事项
- **状态跟踪** - 待办/完成状态管理
- **截止日期** - 设置任务截止时间
- **打卡功能** - 任务完成打卡记录
- **完成率** - 统计任务完成情况
- **历史记录** - 查看任务打卡历史

### 📖 我的日记
- **日记记录** - 每日生活点滴记录
- **心情标记** - emoji心情状态记录
- **天气记录** - 天气状况标记
- **时间管理** - 按日期组织日记
- **月度归档** - 按月份分组展示
- **编辑删除** - 完整的日记管理功能
- **连续记录** - 统计连续写日记天数

### 💾 数据备份
- **一键备份** - 导出所有生活数据
- **数据还原** - 从备份文件恢复数据
- **格式兼容** - 标准JSON格式数据
- **安全保障** - 本地数据存储保护隐私

## 🚀 安装部署

### 环境要求
- Go 1.19+ 
- 现代Web浏览器（Chrome、Firefox、Safari、Edge）

### 快速开始

1. **克隆项目**
```bash
git clone <repository-url>
cd goblog
```

2. **安装依赖**
```bash
go mod tidy
```

3. **运行应用**
```bash
go run main.go
```

4. **访问应用**
打开浏览器访问：http://localhost:8080

### 生产部署

1. **编译应用**
```bash
go build -o life-management main.go
```

2. **运行服务**
```bash
./life-management
```

## 📁 项目结构

```
goblog/
├── main.go              # 主程序入口
├── go.mod              # Go模块定义

│   ├── dashboard.go     # 仪表盘处理器
│   ├── finance.go       # 收支管理处理器
│   ├── habits.go        # 习惯打卡处理器
│   ├── todos.go         # 待办事项处理器
│   ├── diary.go         # 日记管理处理器
│   ├── backup.go        # 备份还原处理器
│   └── categories.go    # 分类管理处理器
├── db/                  # 数据库相关
│   └── db.go           # 数据库初始化和管理
├── models/             # 数据模型
│   └── models.go       # 数据结构定义
├── templates/          # HTML模板
│   ├── layout.html     # 主布局模板
│   ├── dashboard.html  # 仪表盘页面
│   ├── finance.html    # 收支管理页面
│   ├── habits.html     # 习惯打卡页面
│   ├── todos.html      # 待办事项页面
│   ├── diary.html      # 日记管理页面
│   └── backup.html     # 备份还原页面
└── data/              # 数据存储目录

```
